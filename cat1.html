Irrigation Control System

Irrigation Control System ICS-I50-XXX-X is intended for automating drip irrigation and fertigation process in open field, Net-house or Green-house. Depending on crop type and stage of the plant life user can program the quantity of water and fertigation schedule.

User can group together multiple valves in Batches. A common irrigation and fetigation timing can be set for Batches from easy to use Mobile phone App. There are maximum 8 Batches. Series of Batches are grouped together in program. Every morning a existing program is renewed and executed automatically. There are maximum 8 Programs user can program and keep stored in the system.

Features :
Android based easy to use app for configuration and status review
GSM based remote operation
Options for 8, 12, 20 or 28 Valves
System Status indicators(LEDs)
User configurable multiple Programs and Batches
Utility power failure detection and auto adjustment in rest of the valves
Low voltage detection and warning indication
Battery Backedup system allows setup while power is not available
RTC / GSM clock syncing
Auxiliary valve operating voltage detection circuit
Irrigation and Fertigation automation
Main pump and fertigation booster pump control operation
Advance Features :
Fertigation Filter Auto Flush
Controls and Indications :
For both manual and automatic control
8, 12, 20, 28 outputs with 200mA
Main pump and Fertilizer pump start and stop control with indication
Output control status indication
Stable voltage indication
Auto/Manual mode indication
Fertilizer pump status indication
Irrigation Controller

Krushang-500 is a cloud enabled irrigation controller. Farmer can easily schedule and monitor irrigation program through mobile App at his convenience from anywhere. It sends event notification on App. Accurate irrigation will help farmer to increase crop quality and yield. The main objective of this controller is to increase profitability and reduce overhead.

Features :
Time to time crop irrigation
Time to time event notification
Auto start on power restore
Venturi base fertigation
Easy to use mobile application
Cloud base data storage facility
Independent Valve controls - 6 (24 VAC)
Easy to install (Do it yourself kit)
GH-500x Series Green House Controller

InSignEx has design GH-500x Series Controller, to maintain configurable parameter like, temperature, humidity, etc. inside green house. Controller will automatically control the fogger, pad and fan efficiently to achive configured parameter.

It has total 16 configurable output. where user can connect fogger pump, pad pump, fan and solanoid valve as well. Upto 8 Digital Humidity and Temperature sensor can be connect to a single Controller over CAT-05 cable.

Features :
On Panel LCD Display
Keypad for User Interface
Configurable Operation Start and stop time
Configurable Himidity and Temperature Range
Configurable Fogger, Pad and Fan ON and OFF time
Power Quality Monitoring
Auto start on Power Restore
System Live Status On Display
Input voltage
System operation mode
Live Temperature and Humidity
Pump Status Indicator
Low Power
Battery Backup
Advance Features :
Remote update on smart phone
Graphical representation of Temperature and Humidity over period
Remote Alert and notification
WS-2205

WS-2205 Weather Station monitor environmental parameters to improve crop quality and yield.

Why weather Station?

Weather Station notifies farmer soil-moisture-level to schedule precise irrigation. It also identifies perilous condition for plant disease which helps efficient pest-management.

Features :
Soil temperature measurement
Soil moisture measurement
Wind speed and direction
Rain fall measurement
Ambient temperature, moisture and leaf wetness
Wireless monitoring / Solar operated
Cloud base data storage
Easy to install (Do it yourself kit)
